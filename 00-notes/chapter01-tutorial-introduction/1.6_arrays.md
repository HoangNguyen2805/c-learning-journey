# 1.6 - Arrays

## Big idea

This section introduces arrays.

An array lets one variable name store many values.

Instead of creating many separate variables:

```c
int score1;
int score2;
int score3;
int score4;
int score5;
```

we can create one array:

```c
int scores[5];
```

This means:

```text
scores is an array
it can store 5 integer values
```

---

## Array indexes start at 0

If an array has 5 values:

```c
int scores[5];
```

the indexes are:

```text
scores[0]
scores[1]
scores[2]
scores[3]
scores[4]
```

The first value is index `0`.

The last value is index `4`.

So this is wrong:

```c
scores[5]
```

because `scores[5]` would be the sixth value, but the array only has 5 spaces.

---

## Basic example

```c
int scores[5];

scores[0] = 90;
scores[1] = 80;
scores[2] = 70;
scores[3] = 100;
scores[4] = 85;
```

This stores five scores inside one array.

---

## Printing array values

```c
printf("Score 0: %d\n", scores[0]);
printf("Score 1: %d\n", scores[1]);
printf("Score 2: %d\n", scores[2]);
printf("Score 3: %d\n", scores[3]);
printf("Score 4: %d\n", scores[4]);
```

Output:

```text
Score 0: 90
Score 1: 80
Score 2: 70
Score 3: 100
Score 4: 85
```

---

## Array with a for loop

Because array indexes go in order, a loop can print the array.

```c
int i;

for (i = 0; i < 5; i++) {
    printf("Score %d: %d\n", i, scores[i]);
}
```

Meaning:

```text
start i at 0
repeat while i is less than 5
increase i by 1 each time
use i as the array index
```

So the loop prints:

```text
scores[0]
scores[1]
scores[2]
scores[3]
scores[4]
```
