# 1.2 - Variables and Arithmetic Expressions

## Big idea

This section introduces variables and arithmetic expressions using a temperature table.

The program converts Fahrenheit to Celsius.

This section is still a preview. It uses `while`, but loops are not deeply explained yet.

---

## What is a variable?

A variable is a named storage box.

Example:

```c
int fahr;
```

This creates a variable named `fahr`.

It can store an integer value.

---

## Variables used in this section

```c
int fahr;
int celsius;
int lower;
int upper;
int step;
```

Meaning:

```text
fahr    = Fahrenheit temperature
celsius = Celsius temperature
lower   = starting temperature
upper   = ending temperature
step    = how much to increase each time
```

---

## Assigning values

```c
lower = 0;
upper = 300;
step = 20;
fahr = lower;
```

Meaning:

```text
lower starts at 0
upper stops at 300
step increases by 20
fahr starts at lower
```

---

## Arithmetic expression

```c
celsius = 5 * (fahr - 32) / 9;
```

This converts Fahrenheit to Celsius.

The formula is:

```text
Celsius = 5 * (Fahrenheit - 32) / 9
```

Parentheses matter:

```c
(fahr - 32)
```

This part happens first.

---

## Integer division

In this section, the variables are `int`.

So the answer is also an integer.

Example:

```text
5 * (0 - 32) / 9 = -17
```

C removes the decimal part.

So the answer is not exact, but it is good enough for the first example.

---

## The `while` loop preview

```c
while (fahr <= upper) {
    ...
}
```

This means:

```text
Repeat while fahr is less than or equal to upper.
```

Inside the loop, the program prints the temperature, then increases `fahr`.

```c
fahr = fahr + step;
```

This moves to the next Fahrenheit value.

---

## Full program

```c
#include <stdio.h>

int main() {
    int fahr;
    int celsius;
    int lower;
    int upper;
    int step;

    lower = 0;
    upper = 300;
    step = 20;

    fahr = lower;

    printf("Fahrenheit Celsius\n");

    while (fahr <= upper) {
        celsius = 5 * (fahr - 32) / 9;
        printf("%3d %6d\n", fahr, celsius);
        fahr = fahr + step;
    }

    return 0;
}
```
