# 2.7 - Type Conversions

## Big idea

Type conversion means C changes a value from one type to another.

Example:

```c
int a = 10;
double b = a;
```

`a` is an `int`, but it is converted to `double` when stored in `b`.

---

## Automatic conversions

C often converts values automatically.

Example:

```c
double result = 10 + 3.5;
```

`10` is an `int`.

`3.5` is a `double`.

C converts `10` to `10.0`, then does double arithmetic.

---

## Integer division

If both operands are integers, the result is integer division.

```c
10 / 3
```

Result:

```text
3
```

The decimal part is removed.

---

## Decimal division

To get decimal division, at least one operand must be floating-point.

```c
10.0 / 3
```

Result:

```text
3.333...
```

You can also use a cast:

```c
(double) 10 / 3
```

---

## Cast

A cast tells C to convert a value to another type.

Format:

```c
(type) value
```

Example:

```c
(double) a
```

This means:

```text
temporarily treat a as a double
```

Example:

```c
double result = (double) a / b;
```

---

## Character to integer conversion

Characters are stored as integer codes.

Example:

```c
char letter = 'A';
printf("%d\n", letter);
```

This prints the integer code for `'A'`.

This is why character math works:

```c
'7' - '0'
```

Result:

```text
7
```

Because C converts the characters to integer codes first.

---

## Assignment conversions

When you assign a value to a variable, C converts the value to the variable's type.

Example:

```c
double price = 9.99;
int dollars = price;
```

`9.99` becomes `9`.

The decimal part is lost.

Simple rule:

```text
larger/more precise type to smaller type can lose information
```

---

## Usual arithmetic conversions

When two different numeric types are used together, C usually converts the smaller type to the larger or more precise type.

Example:

```c
int a = 10;
double b = 3.5;
double result = a + b;
```

`a` is converted to `double`.

Then C does double arithmetic.

---

## K&R note

K&R also shows examples like converting characters to numbers.

Example idea:

```c
'5' - '0'
```

This changes the character digit `'5'` into the integer value `5`.

K&R also uses functions like `atoi` and character tests later, but the main idea here is type conversion.
