# 2.11 - Conditional Expressions

## Big idea

A conditional expression chooses between two values.

Format:

```c
condition ? value_if_true : value_if_false
```

Example:

```c
max = (a > b) ? a : b;
```

Meaning:

```text
if a > b is true, use a
otherwise, use b
```

---

## The `? :` operator

The conditional operator has three parts:

```c
condition ? expression1 : expression2
```

Meaning:

```text
condition is checked first

if condition is true, the result is expression1

if condition is false, the result is expression2
```

In C:

```text
0 = false
nonzero = true
```

---

## Example: maximum value

```c
int max;
int a = 10;
int b = 20;

max = (a > b) ? a : b;
```

Result:

```text
max = 20
```

---

## Example: minimum value

```c
min = (a < b) ? a : b;
```

Meaning:

```text
if a is less than b, use a
otherwise, use b
```

---

## Conditional expression vs if/else

This:

```c
max = (a > b) ? a : b;
```

is a short expression version of:

```c
if (a > b)
    max = a;
else
    max = b;
```

But in Chapter 2, focus on the expression form:

```c
condition ? true_value : false_value
```

---

## Using it inside printf

A conditional expression can be used directly inside `printf`.

Example:

```c
printf("%s\n", score >= 70 ? "Pass" : "Fail");
```

Meaning:

```text
if score >= 70, print Pass
otherwise, print Fail
```

---

## Only one side is chosen

After the condition is checked, C only uses one result.

```c
condition ? expression1 : expression2
```

If the condition is true, C uses `expression1`.

If the condition is false, C uses `expression2`.

---

## Type note

The two possible results should usually be compatible types.

Good:

```c
max = (a > b) ? a : b;
```

Both choices are integers.

Also good:

```c
printf("%s\n", score >= 70 ? "Pass" : "Fail");
```

Both choices are strings.

Avoid mixing confusing types while learning.

---

## Precedence note

The conditional operator has low precedence.

Use parentheses to make it clear.

Better:

```c
max = (a > b) ? a : b;
```

Harder to read:

```c
max = a > b ? a : b;
```

Full precedence is covered in 2.12.
