# 2.10 - Assignment Operators and Expressions

## Big idea

Assignment stores a value into a variable.

```c
x = 10;
```

Meaning:

```text
put 10 into x
```

---

## Basic assignment operator

```c
=
```

Example:

```c
int x;
x = 10;
```

The left side must be something that can store a value, like a variable.

Correct:

```c
x = 10;
```

Wrong:

```c
10 = x;
```

---

## Assignment expressions have values

In C, an assignment is also an expression.

Example:

```c
y = (x = 5);
```

Meaning:

```text
x gets 5
the expression x = 5 has value 5
y gets 5
```

So after this:

```text
x = 5
y = 5
```

---

## Chained assignment

Because assignment expressions have values, you can write:

```c
a = b = c = 0;
```

This means:

```text
c gets 0
b gets 0
a gets 0
```

Assignment happens right to left.

---

## Compound assignment operators

C has shortcuts for changing a variable.

```text
+=
-=
*=
/=
%=
```

Examples:

```c
x += 5;
x -= 2;
x *= 3;
x /= 2;
x %= 4;
```

Meaning:

```text
x += 5 means x = x + 5
x -= 2 means x = x - 2
x *= 3 means x = x * 3
x /= 2 means x = x / 2
x %= 4 means x = x % 4
```

---

## Bitwise assignment operators

Because K&R already introduced bitwise operators in 2.9, C also has bitwise assignment operators.

```text
&=
|=
^=
<<=
>>=
```

Examples:

```c
x &= mask;
x |= mask;
x ^= mask;
x <<= 1;
x >>= 1;
```

Meaning:

```text
x &= mask means x = x & mask
x |= mask means x = x | mask
x ^= mask means x = x ^ mask
x <<= 1 means x = x << 1
x >>= 1 means x = x >> 1
```

---

## Important compound assignment rule

This:

```c
x *= y + 1;
```

means:

```c
x = x * (y + 1);
```

It does not mean:

```c
x = x * y + 1;
```

So the right side acts like it is grouped.

---

## Why compound assignment is useful

This:

```c
x += 1;
```

is shorter than:

```c
x = x + 1;
```

It also makes it clear that you are updating the same variable.

---

## What I need to remember

```text
= stores a value into a variable

assignment expressions have values

a = b = c = 0 works right to left

+= means add and assign
-= means subtract and assign
*= means multiply and assign
/= means divide and assign
%= means remainder and assign

&= |= ^= <<= >>= are bitwise assignment operators

x *= y + 1 means x = x * (y + 1)

left side of assignment must be a variable or something assignable
```
