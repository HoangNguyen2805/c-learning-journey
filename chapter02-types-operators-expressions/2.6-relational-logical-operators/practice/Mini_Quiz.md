# Mini Quiz - 2.6 Relational and Logical Operators

1. What does a relational operator do?
- Compare but not assign. It compares two values and gives true or false.
2. What does `>` mean?
- Greater than
3. What does `>=` mean?
- Greater or equal to
4. What does `<` mean?
- Less than
5. What does `<=` mean?
- Less than or equal to
6. What does `==` mean?
- Equal to
7. What does `!=` mean?
- Not equal to
8. What is the difference between `=` and `==`?
- = - assign value
- == - compares two values
9. What value does C use for false?
- 0 zero
10. What value does C usually print for true?
- 1 or non-zero value
11. Is any nonzero value considered true?
- Yes
12. What does `&&` mean?
- AND logic
13. What does `||` mean?
- OR logic
14. What does `!` mean?
- NOT logic
15. For `&&`, do both sides need to be true?
- YES both need to be True
16. For `||`, does at least one side need to be true?
- Yes Only one side need to be true
17. What does `!0` give?
- 1 , True
18. What does `!1` give?
- 0 , False
19. What is short-circuit evaluation?
- short-circuit = C stops checking when it already knows the answer
- C stops checking the rest of a logical expression when it already knows the result.
20. For `&&`, what happens if the left side is false?
- Logic end with False . The whole expression is false, so C does not need to check the right side.
21. For `||`, what happens if the left side is true?
- Logic end with True. The whole expression is true, so C does not need to check the right side.
22. What happens first in this expression?

```c
10 + 5 > 12
```
- Addition. 10 + 5 happens first.
23. What is the result of this expression?

```c
10 > 3
```
- True, usually printed as 1.
24. What is the result of this expression?

```c
10 == 3
```
- False, 0.
25. What is the result of this expression?

```c
10 != 3
```
- True, usually printed as 1.
