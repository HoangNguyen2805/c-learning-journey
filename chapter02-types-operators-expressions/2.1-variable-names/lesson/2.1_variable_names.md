# 2.1 - Variable Names

## Big idea

This section explains how names work in C.

Names are used for variables, functions, and symbolic constants.

---

## Valid name characters

A C name can contain:

```text
letters
digits
underscore _
```

Examples:

```c
score
student_age
totalScore
count1
```

A name cannot start with a digit.

Correct:

```c
int score1;
```

Wrong:

```c
int 1score;
```

---

## Underscore

Underscore `_` is allowed in names.

Example:

```c
int student_age;
```

But avoid starting names with `_` because those names may be reserved by the system.

---

## Keywords are reserved

C keywords cannot be used as variable names.

Examples:

```text
int
float
double
char
if
else
while
for
return
void
```

Wrong:

```c
int if;
int float;
```

Keywords are lowercase.

---

## Name length

K&R says at least 31 characters of an internal name are significant.

For external names, old C systems only guaranteed uniqueness for 6 characters and one case.

For modern practice, use clear names that are not too long.

---

## What I need to remember

```text
Names can use letters, digits, and underscore.

Names cannot start with a digit.

Avoid starting names with underscore.

Keywords cannot be used as names.

Use meaningful names.
```
