# 2.8 - Increment and Decrement Operators

## Big idea

C has special operators for adding or subtracting 1.

```text
++ means add 1
-- means subtract 1
```

Examples:

```c
x++;
x--;
```

---

## Increment operator `++`

This:

```c
x++;
```

means:

```c
x = x + 1;
```

Example:

```c
int x = 5;
x++;
```

Now:

```text
x = 6
```

---

## Decrement operator `--`

This:

```c
x--;
```

means:

```c
x = x - 1;
```

Example:

```c
int x = 5;
x--;
```

Now:

```text
x = 4
```

---

## Prefix and postfix

There are two forms:

```c
++x
x++
```

and:

```c
--x
x--
```

If the operator is alone on one line, both forms act the same.

```c
x++;
++x;
```

Both add 1.

---

## Post-increment: `x++`

Post-increment means:

```text
use the old value first
then add 1
```

Example:

```c
int x = 5;
int y = x++;
```

Result:

```text
x = 6
y = 5
```

---

## Pre-increment: `++x`

Pre-increment means:

```text
add 1 first
then use the new value
```

Example:

```c
int x = 5;
int y = ++x;
```

Result:

```text
x = 6
y = 6
```

---

## Post-decrement and pre-decrement

Post-decrement:

```c
int x = 5;
int y = x--;
```

Result:

```text
x = 4
y = 5
```

Pre-decrement:

```c
int x = 5;
int y = --x;
```

Result:

```text
x = 4
y = 4
```

---

## Important rule

`++` and `--` must be used on a variable.

Correct:

```c
x++;
```

Wrong:

```c
5++;
```

You cannot increment a constant value like `5`.

---

## K&R note

K&R uses `++` and `--` a lot with loops, arrays, and strings.

Example idea:

```c
s[j++] = s[i];
```

This means:

```text
use j first
then increase j
```

The full array/string examples belong later, but the operator idea belongs here.

---

## Safe beginner rule

Avoid confusing expressions like:

```c
x = x++ + ++x;
```

This is hard to read and can be unsafe.

Use simple lines instead:

```c
x++;
y = x;
```

---

## What I need to remember

```text
++ adds 1
-- subtracts 1

x++ means use old value, then add 1
++x means add 1, then use new value

x-- means use old value, then subtract 1
--x means subtract 1, then use new value

When alone on one line, x++ and ++x usually do the same thing

++ and -- must be used on variables, not constants

K&R uses these later with arrays and strings
```
