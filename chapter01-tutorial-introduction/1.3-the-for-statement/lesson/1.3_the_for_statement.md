# 1.3 - The For Statement

This section shows how a `while` loop can be rewritten as a `for` loop.

The example prints a Fahrenheit to Celsius table.

The main point is not to teach every loop yet.

The main point is:

```text
A for loop can put the starting value, condition, and update all in one line.
```

---

## Fahrenheit to Celsius Formula

The formula is:

```c
celsius = 5 * (fahr - 32) / 9;
```

Example:

```text
fahr = 100

celsius = 5 * (100 - 32) / 9
celsius = 5 * 68 / 9
celsius = 340 / 9
celsius = 37
```

Because this example uses `int`, the decimal part is removed.

So `37.7` becomes `37`.

---

## Before: while loop version

A `while` loop separates the loop parts.

Example:

```c
int fahr;

fahr = 0;

while (fahr <= 300) {
    printf("%3d %6d\n", fahr, 5 * (fahr - 32) / 9);
    fahr = fahr + 20;
}
```

The three loop parts are in different places:

```text
starting value: fahr = 0;

condition: fahr <= 300

update: fahr = fahr + 20;
```

So the `while` version works like this:

```text
1. Start fahr at 0
2. Check if fahr <= 300
3. Print fahr and celsius
4. Add 20 to fahr
5. Repeat until fahr is greater than 300
```

---

## After: for loop version

A `for` loop puts the starting value, condition, and update in one line.

Example:

```c
int fahr;

for (fahr = 0; fahr <= 300; fahr = fahr + 20) {
    printf("%3d %6d\n", fahr, 5 * (fahr - 32) / 9);
}
```

The `for` loop format is:

```c
for (start; condition; update) {
    // code to repeat
}
```

For this example:

```c
for (fahr = 0; fahr <= 300; fahr = fahr + 20)
```

Breakdown:

```text
fahr = 0
start fahr at 0

fahr <= 300
keep looping while fahr is less than or equal to 300

fahr = fahr + 20
add 20 after each loop
```

---

## while vs for

Both versions print the same result.

### while loop

```c
fahr = 0;

while (fahr <= 300) {
    printf("%3d %6d\n", fahr, 5 * (fahr - 32) / 9);
    fahr = fahr + 20;
}
```

### for loop

```c
for (fahr = 0; fahr <= 300; fahr = fahr + 20) {
    printf("%3d %6d\n", fahr, 5 * (fahr - 32) / 9);
}
```

The `for` loop is cleaner here because the loop control is all in one line.

---

## printf field width review

The example uses:

```c
printf("%3d %6d\n", fahr, celsius);
```

Meaning:

```text
%3d = print an integer using at least 3 spaces wide

%6d = print an integer using at least 6 spaces wide
```

This makes the output line up like a table.

Example:

```text
  0    -17
 20     -6
100     37
```

## Complete lesson code

```c
#include <stdio.h>

int main() {
    int fahr;

    printf("--- Before: while loop version ---\n");
    printf("Fahrenheit Celsius\n");

    fahr = 0;

    while (fahr <= 300) {
        printf("%3d %6d\n", fahr, 5 * (fahr - 32) / 9);
        fahr = fahr + 20;
    }

    printf("\n--- After: for loop version ---\n");
    printf("Fahrenheit Celsius\n");

    for (fahr = 0; fahr <= 300; fahr = fahr + 20) {
        printf("%3d %6d\n", fahr, 5 * (fahr - 32) / 9);
    }

    return 0;
}
```

---

## Memory rule

```text
while loop:
start before the loop
condition in the while line
update inside the loop body

for loop:
start, condition, and update are all in one line
```

The `for` loop format:

```c
for (start; condition; update) {
    // repeated code
}
```
