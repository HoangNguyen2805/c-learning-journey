# 1.7 - Functions

## Big idea

This section introduces functions.

A function is a reusable block of code.

Instead of writing the same code many times, we can put that code inside a function and call it when we need it.

---

## K&R example idea

K&R uses a function named `power`.

The function calculates:

```text
base raised to n
```

Example:

```text
power(2, 3) = 8
```

because:

```text
2 * 2 * 2 = 8
```

---

## Function prototype

```c
int power(int base, int n);
```

This tells C:

```text
There is a function named power.
It takes two int values.
It returns an int value.
```

This line appears before `main` so `main` knows the function exists.

---

## Calling a function

Inside `main`, we call the function:

```c
power(2, i)
```

This means:

```text
Use 2 as the base.
Use i as the exponent.
Return the answer.
```

Example:

```c
printf("%d\n", power(2, 3));
```

Output:

```text
8
```

---

## Function definition

```c
int power(int base, int n) {
    int i;
    int result;

    result = 1;

    for (i = 1; i <= n; i++) {
        result = result * base;
    }

    return result;
}
```

This is the actual function body.

It explains what the function does.

---

## Parameters

```c
int power(int base, int n)
```

`base` and `n` are parameters.

They receive values when the function is called.

Example:

```c
power(2, 3)
```

Inside the function:

```text
base = 2
n = 3
```

---

## Return value

```c
return result;
```

This sends the answer back to the place where the function was called.

Example:

```c
power(2, 3)
```

returns:

```text
8
```

---

## Full program

```c
#include <stdio.h>

int power(int base, int n);

int main() {
    int i;

    for (i = 0; i < 10; i++) {
        printf("%d %d %d\n", i, power(2, i), power(-3, i));
    }

    return 0;
}

int power(int base, int n) {
    int i;
    int result;

    result = 1;

    for (i = 1; i <= n; i++) {
        result = result * base;
    }

    return result;
}
```

---

## Output meaning

Each line prints:

```text
i   2^i   (-3)^i
```

Example:

```text
3 8 -27
```

Meaning:

```text
i = 3
2^3 = 8
(-3)^3 = -27
```
