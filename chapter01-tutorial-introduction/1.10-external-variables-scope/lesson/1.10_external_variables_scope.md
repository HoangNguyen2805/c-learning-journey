# 1.10 - External Variables and Scope

## Big idea

This section introduces external variables and scope.

An external variable is a variable created outside all functions.

Because it is outside all functions, multiple functions can use it.

---

## Local variable

A local variable is created inside a function.

Example:

```c
int main() {
    int number;
}
```

`number` belongs to `main`.

Other functions cannot directly use it.

---

## External variable

An external variable is created outside all functions.

Example:

```c
int count;

int main() {
    count = 0;
}
```

`count` is external because it is outside `main`.

Other functions can use it too.

---

## Example program

```c
#include <stdio.h>

int count;

void addOne(void);

int main() {
    count = 0;

    printf("Before function calls: %d\n", count);

    addOne();
    addOne();
    addOne();

    printf("After function calls: %d\n", count);

    return 0;
}

void addOne(void) {
    count = count + 1;

    printf("Inside addOne: %d\n", count);
}
```

---

## Output

```text
Before function calls: 0
Inside addOne: 1
Inside addOne: 2
Inside addOne: 3
After function calls: 3
```

---

## Why does `count` keep changing?

`count` is external.

That means both `main` and `addOne` can use the same `count`.

When `addOne` changes `count`, the change is still there after the function ends.

---

## Scope

Scope means where a variable can be used.

A local variable has local scope.

An external variable has wider scope.
