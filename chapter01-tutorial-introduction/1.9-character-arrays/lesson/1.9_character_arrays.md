# 1.9 - Character Arrays

## Big idea

This section introduces character arrays.

A character array is an array that stores characters.

In C, strings are stored as character arrays.

---

## Character array example

```c
char name[] = "Hoang";
```

This stores the text `"Hoang"` inside a character array.

The array contains:

```text
name[0] = 'H'
name[1] = 'o'
name[2] = 'a'
name[3] = 'n'
name[4] = 'g'
```

---

## Strings end with `\0`

C strings end with a special character:

```c
'\0'
```

This is called the null character.

It marks the end of the string.

So this:

```c
char name[] = "Hoang";
```

is really stored like this:

```text
'H' 'o' 'a' 'n' 'g' '\0'
```

That means the array needs 6 spaces, not just 5.

---

## Printing a string

Use `%s` to print a string.

```c
printf("%s\n", name);
```

Output:

```text
Hoang
```

`printf` keeps printing characters until it finds `'\0'`.

---

## Printing one character

Use `%c` to print one character.

```c
printf("%c\n", name[0]);
```

Output:

```text
H
```

---

## Full example

```c
#include <stdio.h>

int main() {
    char name[] = "Hoang";

    printf("Name as string: %s\n", name);

    printf("Characters:\n");
    printf("%c\n", name[0]);
    printf("%c\n", name[1]);
    printf("%c\n", name[2]);
    printf("%c\n", name[3]);
    printf("%c\n", name[4]);

    return 0;
}
```
