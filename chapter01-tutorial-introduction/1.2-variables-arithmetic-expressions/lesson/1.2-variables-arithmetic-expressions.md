# 1.2 Variables and Arithmetic Expressions

## The Big Picture

In 1.1 your program only printed a fixed string. Now it can **compute**.

This section introduces six new ideas at once:
comments, declarations, variables, arithmetic expressions, loops, and formatted output.

---

## The Program K&R Builds

Convert Fahrenheit to Celsius using `C = (5/9)(F - 32)`:

```c
#include <stdio.h>

/* print Fahrenheit-Celsius table
   for fahr = 0, 20, ..., 300 */
main()
{
    int fahr, celsius;
    int lower, upper, step;

    lower = 0;      /* lower limit of temperature scale */
    upper = 300;    /* upper limit */
    step = 20;      /* step size */

    fahr = lower;
    while (fahr <= upper) {
        celsius = 5 * (fahr - 32) / 9;
        printf("%d\t%d\n", fahr, celsius);
        fahr = fahr + step;
    }
}
```

Everything below explains a piece of this program.

---

## Comments

```c
/* print Fahrenheit-Celsius table
   for fahr = 0, 20, ..., 300 */
```

Anything between `/*` and `*/` is ignored by the compiler. Comments may appear
anywhere a blank, tab, or newline can. Use them to say *why*, not *what* —
the code already says what.

---

## Declarations

**In C, all variables must be declared before they are used**, usually at the
beginning of the function before any executable statements.

```c
int fahr, celsius;
int lower, upper, step;
```

A declaration announces the properties of variables: a type, then a list of names.

`int` means the variables are integers. `float` means floating point — numbers
that may have a fractional part.

### Other basic types K&R mentions here

| Type | Meaning |
|------|---------|
| `char` | character — a single byte |
| `short` | short integer |
| `long` | long integer |
| `double` | double-precision floating point |

Sizes are machine-dependent. 16-bit `int` (range -32768 to +32767) and 32-bit
`int` are both common. A `float` is typically 32 bits with at least six
significant digits.

---

## Assignment

```c
lower = 0;
upper = 300;
step = 20;
```

These set the variables to their initial values. `=` means *assign*, not
*equals*. Individual statements are terminated by semicolons.

---

## The while Loop

```c
while (fahr <= upper) {
    ...
}
```

How it operates:

1. The condition in parentheses is tested.
2. If true, the body of the loop is executed.
3. The condition is retested; if still true, the body runs again.
4. When the test becomes false, the loop ends and execution continues after it.

The body can be one or more statements in braces, or a single statement
without braces:

```c
while (i < j)
    i = 2 * i;
```

**Always indent the statements controlled by the `while`.** C compilers don't
care how a program looks, but proper indentation is critical for people
reading it. Write one statement per line and use blanks around operators.

---

## Arithmetic, and Why 5/9 Is a Trap

```c
celsius = 5 * (fahr - 32) / 9;
```

Why multiply by 5 first and divide by 9 second, instead of writing `5/9`?

**Because in C, integer division truncates: any fractional part is discarded.**

Since 5 and 9 are integers, `5/9` would be truncated to zero, and every Celsius
temperature would print as 0.

```c
5 / 9        /* 0   — both operands int */
5.0 / 9.0    /* 0.555...  — both operands float */
5.0 / 9      /* 0.555...  — 9 is converted to float */
```

If an arithmetic operator has integer operands, an integer operation is
performed. If it has one floating-point operand and one integer operand, the
integer is converted to floating point before the operation.

## Formatted Output

`printf` is a general-purpose output formatting function. Its first argument is
a string to be printed, with each `%` marking where one of the later arguments
is substituted, and in what form.

```c
printf("%d\t%d\n", fahr, celsius);
```

This prints the two integers with a tab between them.

**Each `%` is paired with the corresponding argument by position. They must
match up by number and type, or you will get wrong answers.**

### Conversion specifications

| Spec | Meaning |
|------|---------|
| `%d` | print as decimal integer |
| `%6d` | print as decimal integer, at least 6 characters wide |
| `%f` | print as floating point |
| `%6f` | print as floating point, at least 6 characters wide |
| `%.2f` | print as floating point, 2 characters after decimal point |
| `%6.2f` | at least 6 wide, 2 after the decimal point |

`printf` also recognizes `%o` for octal, `%x` for hexadecimal, `%c` for
character, `%s` for character string, and `%%` for a literal `%`.

Width and precision may each be omitted.

Note: `printf` is not part of the C language. There is no input or output
defined in C itself; `printf` is a useful function from the standard library.

## The Floating-Point Version

Integer arithmetic makes the table inaccurate — 0°F is about -17.8°C, not -17.
Using `float` fixes it:

```c
#include <stdio.h>

/* print Fahrenheit-Celsius table
   for fahr = 0, 20, ..., 300; floating-point version */
main()
{
    float fahr, celsius;
    float lower, upper, step;

    lower = 0;
    upper = 300;
    step = 20;

    fahr = lower;
    while (fahr <= upper) {
        celsius = (5.0/9.0) * (fahr - 32.0);
        printf("%3.0f %6.1f\n", fahr, celsius);
        fahr = fahr + step;
    }
}
```

`%3.0f` — at least three characters wide, no decimal point, no fraction digits.
`%6.1f` — at least six characters wide, one digit after the decimal point.

Writing floating-point constants with explicit decimal points, even when the
value is integral (`32.0` rather than `32`), emphasizes their floating-point
nature to human readers.

## Common Mistakes

### Integer division where you needed a fraction
```c
float celsius = 5 / 9 * (fahr - 32);   /* 5/9 is 0, so celsius is 0 */
```
Fix: `(5.0 / 9.0) * (fahr - 32.0)`

### Forgetting to advance the loop variable
```c
while (fahr <= upper) {
    printf("%d\n", fahr);
    /* no fahr = fahr + step  ->  runs forever */
}
```

### Mismatched conversion specification
```c
float price = 9.99;
printf("%d\n", price);   /* %d with a float argument — wrong answer */
```
Fix: `printf("%f\n", price);`

### Using a variable before assigning it
```c
int x;
printf("%d\n", x);   /* x holds whatever was in that memory */
```

## Summary

| Concept | Form |
|---------|------|
| Comment | `/* ... */` |
| Declaration | `int fahr, celsius;` |
| Assignment | `lower = 0;` |
| Integer division | `5 / 9` is `0` |
| Float division | `5.0 / 9.0` is `0.555...` |
| while loop | `while (cond) { ... }` |
| Print integer | `printf("%d", n);` |
| Print float, 1 decimal | `printf("%6.1f", x);` |
| Tab / newline | `\t` / `\n` |
