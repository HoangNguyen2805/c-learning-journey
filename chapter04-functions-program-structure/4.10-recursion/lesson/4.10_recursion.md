# 4.10 - Recursion

## Big idea

Recursion means a function calls itself.

Example idea:

```text
function calls itself
then eventually stops
```

A recursive function needs two parts:

```text
base case = when to stop

recursive case = when the function calls itself again
```

---

## Simple recursion example

```c
void countdown(int n) {
    if (n <= 0) {
        printf("Done\n");
    }
    else {
        printf("%d\n", n);
        countdown(n - 1);
    }
}
```

If we call:

```c
countdown(3);
```

Output:

```text
3
2
1
Done
```

---

## Base case

The base case stops the recursion.

```c
if (n <= 0) {
    printf("Done\n");
}
```

Without a base case, the function keeps calling itself forever.

That can crash the program.

---

## Recursive case

The recursive case makes progress toward the base case.

```c
countdown(n - 1);
```

Each call uses a smaller number.

```text
countdown(3)
countdown(2)
countdown(1)
countdown(0)
stop
```

---

## K&R printd example

K&R uses recursion to print an integer one digit at a time.

Idea:

```text
print the front digits first
then print the last digit
```

Example:

```text
printd(123)
```

Process:

```text
printd(12)
printd(1)
print '1'
print '2'
print '3'
```

Output:

```text
123
```

---

## Why recursion works

Each recursive call gets its own copy of local variables and parameters.

So this:

```c
printd(123);
```

creates calls like:

```text
printd(123)
printd(12)
printd(1)
```

Then they finish in reverse order.

---

## Recursion and the call stack

Every function call is saved until it finishes.

This is called the call stack.

For recursion:

```text
each recursive call is added to the stack
when the base case is reached, calls return back one by one
```

---

## Factorial example

Factorial means:

```text
5! = 5 * 4 * 3 * 2 * 1
```

Recursive idea:

```text
factorial(n) = n * factorial(n - 1)
```

Base case:

```text
factorial(0) = 1
```

---

## Recursion can be useful

Recursion is useful when a problem naturally breaks into smaller versions of itself.

Examples:

```text
printing digits
factorial
tree problems
sorting problems like quicksort
```

K&R also mentions quicksort as a recursive example.
