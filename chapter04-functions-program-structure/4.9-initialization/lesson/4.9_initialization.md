# 4.9 - Initialization

## Big idea

Initialization means giving a variable its starting value when it is created.

Example:

```c
int x = 10;
```

This means:

```text
create x and start it with 10
```

---

## Declaration without initialization

Example:

```c
int x;
```

This creates `x`, but does not give it a clear starting value if it is a local automatic variable.

A local variable without initialization may contain garbage.

Bad idea:

```c
int x;
printf("%d\n", x);
```

You should initialize local variables before using them.

---

## Declaration with initialization

Example:

```c
int x = 10;
double price = 19.99;
char grade = 'A';
```

This is better because the variable starts with a known value.

---

## Automatic local variables

A normal local variable inside a function is automatic.

Example:

```c
int main() {
    int x;
    return 0;
}
```

If a local automatic variable is not initialized, its value is unknown.

Rule:

```text
local automatic variable without initialization = garbage value
```

---

## External and static variables

External variables and static variables are automatically initialized to zero if you do not give them a value.

Example:

```c
int externalCount;

int main() {
    static int staticCount;

    printf("%d\n", externalCount);
    printf("%d\n", staticCount);

    return 0;
}
```

Both start as `0`.

Rule:

```text
external variable without initialization = 0

static variable without initialization = 0

automatic local variable without initialization = unknown garbage
```

---

## Static local variable initialization

A static local variable is initialized only once.

Example:

```c
void countCalls(void) {
    static int count = 0;

    count++;
    printf("%d\n", count);
}
```

The `count = 0` happens only one time.

After that, `count` remembers its value.

---

## Array initialization

Arrays can be initialized with braces.

Example:

```c
int numbers[5] = {10, 20, 30, 40, 50};
```

This means:

```text
numbers[0] = 10
numbers[1] = 20
numbers[2] = 30
numbers[3] = 40
numbers[4] = 50
```

---

## Partial array initialization

Example:

```c
int numbers[5] = {10, 20};
```

This means:

```text
numbers[0] = 10
numbers[1] = 20
numbers[2] = 0
numbers[3] = 0
numbers[4] = 0
```

Missing values become `0`.

---

## Character array initialization

A character array can be initialized with characters:

```c
char word[] = {'h', 'i', '\0'};
```

Or with a string literal:

```c
char word[] = "hi";
```

These mean almost the same thing.

The string version automatically adds `'\0'`.

```text
"hi" = 'h', 'i', '\0'
```

---

## External/static initialization rule

External and static variables must be initialized with constant values.

Example:

```c
static int max = 100;
```

This is okay.

For now, just remember:

```text
external and static variables are initialized before the program starts
```
