# Chapter 5.5 — Character Pointers and Functions

## Overview

This chapter explores two ways to work with strings in C:
1. **String arrays** — fixed-size, mutable
2. **String pointers** — point to string literals, read-only

And functions that operate on strings using pointers: `strcmp`, `strcpy`, `strlen`.

## Part 1: String Arrays vs String Pointers

### String Array (amessage)

```c
char amessage[] = "now is the time";  // array of chars
```

**Properties:**
- **Mutable** — you can modify individual characters
- **Fixed size** — allocated at compile time
- **On stack** — local scope

**Example:**
```c
char amessage[] = "hello";
amessage[0] = 'H';           // OK — change 'h' to 'H'
printf("%s\n", amessage);    // prints: "Hello"
```

### String Pointer (pmessage)

```c
char *pmessage = "now is the time";  // pointer to string literal
```

**Properties:**
- **Read-only** — you cannot modify (string is in read-only memory)
- **Dynamic size** — can point to strings of any length
- **Points to literal** — stored in program memory

**Example:**
```c
char *pmessage = "hello";
pmessage[0] = 'H';           // ERROR — undefined behavior (read-only)
printf("%s\n", pmessage);    // prints: "hello"
```

### Key Difference

| Aspect | Array `char a[]` | Pointer `char *p` |
|--------|------------------|-------------------|
| **Mutability** | Can modify characters | Cannot modify (read-only) |
| **Size** | Fixed at compile time | Can point to any size |
| **Memory** | Stack | String literal in program memory |
| **Use case** | Input buffer, mutable strings | Constant strings, literals |

## Part 2: String Functions Using Pointers

### strlen() — String Length

```c
int strlen(char *s) {
    int len = 0;
    while (*s != '\0') {
        len++;
        s++;
    }
    return len;
}
```

**How it works:**
1. Start pointer at beginning
2. Loop while character is not null terminator
3. Count each character, increment pointer
4. Return count

**Example:**
```c
char *str = "hello";
printf("%d\n", strlen(str));  // prints: 5
```

### strcpy() — String Copy

```c
void strcpy(char *s, char *t) {
    while (*s++ = *t++)
        ;  // copy until null terminator
}
```

**How it works:**
1. Dereference `t` (get source character)
2. Assign to dereferenced `s` (put in destination)
3. Increment both pointers
4. Loop continues until `*t` is `\0` (null copied, loop stops)

**Breaking down `*s++ = *t++`:**
- `*t++` — get value at t, then increment t pointer (not the char value)
- `*s++` — assign to s, then increment s pointer
- When `*t` is `\0`, the assignment happens, then loop stops

**Example:**
```c
char source[] = "hello";
char dest[10];
strcpy(dest, source);
printf("%s\n", dest);  // prints: "hello"
```

### strcmp() — String Compare

```c
int strcmp(char *s, char *t) {
    while (*s == *t) {
        if (*s == '\0')
            return 0;  // equal
        s++;
        t++;
    }
    return *s - *t;  // return difference
}
```

**How it works:**
1. Loop while characters match
2. If null terminator reached, strings are equal → return 0
3. If characters differ → return difference (*s - *t)

**Example:**
```c
char *s1 = "hello";
char *s2 = "hello";
char *s3 = "world";

printf("%d\n", strcmp(s1, s2));  // prints: 0 (equal)
printf("%d\n", strcmp(s1, s3));  // prints: negative (s1 < s3)
```

## Part 3: Reading Strings with Spaces

### Problem with scanf("%s", str)

```c
scanf("%s", str);  // stops at whitespace
```

If user enters "hello world", only "hello" is read.

### Solution: scanf with format

```c
scanf("%[^\n]", str);  // read until newline (includes spaces)
```

**How it works:**
- `[^\n]` — character class meaning "anything except newline"
- Reads all characters until newline, **including spaces**

**Example:**
```c
char str[50];
printf("Enter your name: ");
scanf("%[^\n]", str);
printf("You entered: %s\n", str);  // can now handle spaces
```

## Part 4: Pointer Arithmetic with Strings

### Advancing Through a String

```c
char *s = "hello";

printf("%c\n", *s);      // 'h'
printf("%c\n", *(s+1));  // 'e'
printf("%c\n", *(s+2));  // 'l'
```

Or increment pointer:
```c
char *s = "hello";
while (*s != '\0') {
    printf("%c", *s);
    s++;  // move to next character
}
```

### Copying Character by Character

```c
char source[] = "world";
char dest[10];
char *s = source;
char *d = dest;

while ((*d++ = *s++) != '\0')
    ;  // copy until null
```

This loop:
1. Copies current char from source to dest
2. Increments both pointers
3. Stops when null terminator copied

## Part 5: Common String Operations

### Finding a Character

```c
char *strchr(char *s, char c) {
    while (*s != c)
        if (*s++ == '\0')
            return NULL;  // not found
    return s;  // found
}
```

### Copying N Characters

```c
void strncpy(char *s, char *t, int n) {
    while (n-- > 0)
        *s++ = *t++;
}
```

## Summary

| Concept | Key Point |
|---------|-----------|
| **Array** | Fixed size, mutable, stack |
| **Pointer** | Variable size, read-only literals, points to program memory |
| **strlen** | Count characters until null terminator |
| **strcpy** | Copy string using `while (*s++ = *t++)` |
| **strcmp** | Compare loop until difference or null |
| **scanf with spaces** | Use `%[^\n]` instead of `%s` |
| **Pointer arithmetic** | `*s++` — dereference, then increment pointer |
