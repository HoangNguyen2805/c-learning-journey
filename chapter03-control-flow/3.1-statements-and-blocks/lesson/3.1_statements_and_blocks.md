# 3.1 - Statements and Blocks

## Big idea

A statement is one complete instruction in C.

Most simple statements end with a semicolon.

```c
x = 5;
printf("%d\n", x);
```

---

## Expression statements

An expression becomes a statement when you add a semicolon.

```c
x = 10;
x = x + 1;
printf("Hello\n");
```

The semicolon tells C:

```text
this instruction is finished
```

---

## Blocks

A block groups multiple statements inside braces.

```c
{
    x = 5;
    y = 10;
    printf("%d\n", x + y);
}
```

A block acts like one statement.

This is important for `if`, `while`, and `for`.

---

## No semicolon after a block

A simple statement needs a semicolon:

```c
x = 5;
```

A block does not need a semicolon after the closing brace:

```c
{
    x = 5;
    y = 10;
}
```
