# Mini Quiz - 3.1 Statements and Blocks

1. What is a statement in C?
- A statement is one complete instruction for C to do.
2. What symbol ends most simple statements?
- ;
3. What does a semicolon tell C?
- It tells C that the statement is finished.
4. What is a block?
- A block is a group of statements inside braces { }.
5. What symbols are used to create a block?
- { }
6. Can a block contain multiple statements?
- Yes
7. Does a normal block need a semicolon after the closing brace?
- No
8. Why are blocks useful?
- Blocks let multiple statements act together as one unit.
9. What later control-flow tools use blocks?
- if/else, switch, loops, and functions.
10. What is the main idea of K&R 3.1?
- C programs are made of statements, and blocks group statements together using { }.
