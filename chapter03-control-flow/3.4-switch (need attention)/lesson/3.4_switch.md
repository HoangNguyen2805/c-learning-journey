# 3.4 - Switch

## Big idea

`switch` chooses what code to run based on one exact value.

It is useful for things like:

```text id="98vuvl"
menu choices
letter grades
day numbers
simple commands
```

Example:

```c id="tqc6fx"
switch (choice) {
    case 1:
        printf("Start\n");
        break;

    case 2:
        printf("Settings\n");
        break;

    default:
        printf("Invalid\n");
}
```

---

## `switch`

```c id="7j1qgh"
switch (choice)
```

means:

```text id="08rd1u"
check the value of choice
```

---

## `case`

```c id="68fzkt"
case 1:
```

means:

```text id="it0ngi"
if choice equals 1, start here
```

`case` values must be constant values.

Examples:

```c id="9ao3ey"
case 1:
case 'A':
case 100:
```

---

## `break`

`break` stops the switch.

```c id="kqk9z4"
break;
```

Without `break`, C keeps going into the next case.

This is called fall-through.

Beginner rule:

```text id="v10ew1"
Put break after each case unless you purposely want fall-through.
```

---

## `default`

`default` runs when no case matches.

```c id="6hb28v"
default:
    printf("Invalid choice\n");
```

It is like the final `else`.

---

## Switch vs else-if

Use `switch` for exact values.

```c id="38c6n4"
choice == 1
choice == 2
choice == 3
```

Use `else if` for ranges or complex conditions.

```c id="hfs3gl"
score >= 90
score >= 80
age >= 18 && hasID
```

Simple rule:

```text id="m1z71f"
switch = exact values
else-if = ranges and conditions
```

---

## Fall-through example

If there is no `break`, C keeps running.

```c id="at36wo"
switch (choice) {
    case 1:
        printf("One\n");
    case 2:
        printf("Two\n");
}
```

If `choice` is `1`, output is:

```text id="12g8dy"
One
Two
```

Usually this is a mistake.
