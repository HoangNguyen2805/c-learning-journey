# 3.2 - If-Else

## Big idea

`if` lets a program make a decision.

```c
if (condition)
    statement;
```

Meaning:

```text
if the condition is true, run the statement
if the condition is false, skip the statement
```

In C:

```text
0 = false
nonzero = true
```

---

## If statement

Example:

```c
if (score >= 70)
    printf("Pass\n");
```

If `score >= 70` is true, the message prints.

If it is false, nothing happens.

---

## If-else statement

`else` gives the program another choice.

```c
if (score >= 70)
    printf("Pass\n");
else
    printf("Fail\n");
```

Meaning:

```text
if score is at least 70, print Pass
otherwise, print Fail
```

---

## Use braces for multiple statements

A block lets one `if` control multiple statements.

```c
if (score >= 70) {
    printf("Pass\n");
    printf("Good job\n");
}
```

Without braces, only the next statement belongs to the `if`.

---

## Nested if

An `if` can be inside another `if`.

```c
if (age >= 18) {
    if (hasID)
        printf("You may enter\n");
    else
        printf("You need ID\n");
}
else {
    printf("Too young\n");
}
```

Nested `if` statements are useful, but they can become hard to read.

Use braces to make the meaning clear.

---

## Dangling else

K&R warns about this problem:

```c
if (n > 0)
    if (a > b)
        z = a;
else
    z = b;
```

The `else` belongs to the nearest previous `if` that does not already have an `else`.

So it actually means:

```c
if (n > 0)
    if (a > b)
        z = a;
    else
        z = b;
```

Better:

```c
if (n > 0) {
    if (a > b)
        z = a;
    else
        z = b;
}
```

Best beginner rule:

```text
Use braces when there is any chance of confusion.
```

---

## Common mistake: semicolon after if

Wrong:

```c
if (score >= 70);
    printf("Pass\n");
```

The semicolon ends the `if` early.

Correct:

```c
if (score >= 70)
    printf("Pass\n");
```

## What I need to remember

```text
if = run code only when condition is true

else = run different code when condition is false

0 = false
nonzero = true

use braces for multiple statements

else matches the nearest unmatched if

do not put a semicolon right after if(condition)

use braces to avoid confusing nested if statements
```
