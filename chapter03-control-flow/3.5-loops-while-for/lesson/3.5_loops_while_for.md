# K&R 3.5 - Loops: `while` and `for`

## Goal

This lesson explains how C repeats code using loops.

By the end of this lesson, you should understand:

* why loops exist
* how `while` works
* how `for` works
* when to use `while`
* when to use `for`
* how loop variables change
* how infinite loops happen
* how `%` can help with even/odd checks
* how loops connect with `if`, `else-if`, and `switch`

---

# 1. Why loops exist

Sometimes you need to repeat the same kind of action many times.

Without a loop:

```c
printf("1\n");
printf("2\n");
printf("3\n");
printf("4\n");
printf("5\n");
```

This works, but it is bad if you need to print 1 to 100.

With a loop:

```c
int i;

for (i = 1; i <= 5; i++)
{
   printf("%d\n", i);
}
```

Output:

```text
1
2
3
4
5
```

The loop repeats the `printf` for you.

---

# 2. The main idea of a loop

A loop usually has 3 important parts:

```text
1. Start value
2. Condition
3. Update
```

Example:

```c
int i = 1;

while (i <= 5)
{
   printf("%d\n", i);
   i++;
}
```

Breakdown:

```text
Start:     i = 1
Condition: i <= 5
Update:    i++
```

The loop keeps running while the condition is true.

# 3. `while` loop format

Full format:

```c
while (condition)
{
   statement1;
   statement2;
}
```

Example:

```c
#include <stdio.h>

int main()
{
   int i = 1;

   while (i <= 5)
   {
      printf("%d\n", i);
      i++;
   }

   return 0;
}
```

Output:

```text
1
2
3
4
5
```

---

# 4. How `while` works step by step

Code:

```c
int i = 1;

while (i <= 3)
{
   printf("%d\n", i);
   i++;
}
```

Step-by-step:

```text
i = 1
Check: 1 <= 3 true
Print 1
i becomes 2

Check: 2 <= 3 true
Print 2
i becomes 3

Check: 3 <= 3 true
Print 3
i becomes 4

Check: 4 <= 3 false
Loop stops
```

Output:

```text
1
2
3
```

---

# 5. Important rule: the condition is checked before the loop body

In a `while` loop, C checks the condition first.

Example:

```c
int i = 10;

while (i <= 5)
{
   printf("%d\n", i);
   i++;
}
```

Output:

```text
No output
```

Why?

Because:

```text
10 <= 5 is false
```

So the loop body never runs.

---

# 6. Infinite loop

An infinite loop happens when the condition never becomes false.

Bad example:

```c
int i = 1;

while (i <= 5)
{
   printf("%d\n", i);
}
```

Problem:

```text
i never changes
```

So `i` stays `1` forever.

Correct version:

```c
int i = 1;

while (i <= 5)
{
   printf("%d\n", i);
   i++;
}
```

The update `i++` is needed.

---

# 7. `i++`

This:

```c
i++;
```

means:

```c
i = i + 1;
```

Example:

```c
int i = 5;
i++;
```

Now `i` is:

```text
6
```

---

# 8. Counting down with `while`

You can also count down.

```c
#include <stdio.h>

int main()
{
   int i = 5;

   while (i >= 1)
   {
      printf("%d\n", i);
      i--;
   }

   return 0;
}
```

Output:

```text
5
4
3
2
1
```

This:

```c
i--;
```

means:

```c
i = i - 1;
```

# 9. `for` loop format

A `for` loop puts the start, condition, and update in one line.

Full format:

```c
for (initialization; condition; update)
{
   statement1;
   statement2;
}
```

Example:

```c
#include <stdio.h>

int main()
{
   int i;

   for (i = 1; i <= 5; i++)
   {
      printf("%d\n", i);
   }

   return 0;
}
```

Output:

```text
1
2
3
4
5
```

---

# 10. How `for` works

Code:

```c
for (i = 1; i <= 3; i++)
{
   printf("%d\n", i);
}
```

Step-by-step:

```text
1. i = 1 happens first
2. check i <= 3
3. run loop body
4. run i++
5. check i <= 3 again
6. repeat until condition is false
```

Detailed:

```text
i = 1
1 <= 3 true
print 1
i++ makes i = 2

2 <= 3 true
print 2
i++ makes i = 3

3 <= 3 true
print 3
i++ makes i = 4

4 <= 3 false
stop
```

# 11. `while` vs `for`

Both can do the same job.

## `while`

Good when you do not know exactly how many times the loop will run.

Example idea:

```text
Keep asking until input is valid.
Keep running while user says yes.
Keep reading until end of file.
```

Example:

```c
while (choice != 0)
{
   printf("Menu runs again\n");
}
```

## `for`

Good when you know the start and end.

Example idea:

```text
Print 1 to 10.
Add numbers from start to end.
Run exactly 5 times.
```

Example:

```c
for (i = 1; i <= 10; i++)
{
   printf("%d\n", i);
}
```

## `while`

```c
int i = 1;

   printf("%d\n", i);
   i++;

```

## `for`

```c
int i;

for (i = 1; i <= 5; i++)
{
   printf("%d\n", i);

```

Both print:

```text
1
2
3
4
5
```

# 13. Common beginner mistake: semicolon after loop condition

Bad:

```c
int i = 1;

while (i <= 5);
{
   printf("%d\n", i);
   i++;
}
```

The semicolon after `while (i <= 5)` is a problem.

This:

```c
while (i <= 5);
```

means the loop body is empty.

Correct:

```c
while (i <= 5)
{
   printf("%d\n", i);
   i++;
}
```

Same issue with `for`.

Bad:

```c
for (i = 1; i <= 5; i++);
{
   printf("%d\n", i);
}
```

Correct:

```c
for (i = 1; i <= 5; i++)
{
   printf("%d\n", i);
}
```

# 14. Loop with `scanf`

You can use input to control a loop.

Example:

```c
#include <stdio.h>

int main()
{
   int count;
   int i;

   printf("How many times? ");
   scanf("%d", &count);

   for (i = 1; i <= count; i++)
   {
      printf("Hello\n");
   }

   return 0;
}
```

If the user enters:

```text
3
```

Output:

```text
Hello
Hello
Hello
```

# 15. Sum with a loop

A loop can add numbers.

Example: add numbers from 1 to 5.

```c
#include <stdio.h>

int main()
{
   int i;
   int sum = 0;

   for (i = 1; i <= 5; i++)
   {
      sum = sum + i;
   }

   printf("Sum: %d\n", sum);

   return 0;
}
```

Step-by-step:

```text
sum = 0

i = 1 → sum = 0 + 1 = 1
i = 2 → sum = 1 + 2 = 3
i = 3 → sum = 3 + 3 = 6
i = 4 → sum = 6 + 4 = 10
i = 5 → sum = 10 + 5 = 15
```

Output:

```text
Sum: 15
```

# 16. Even and odd numbers

Use the remainder operator:

```c
%
```

Example:

```c
10 % 2
```

Result:

```text
0
```

Because 10 divides evenly by 2.

Example:

```c
11 % 2
```

Result:

```text
1
```

Because 11 has a remainder of 1 when divided by 2.

So:

```c
if (i % 2 == 0)
{
   printf("Even\n");
}
else
{
   printf("Odd\n");
}
```

# 19. Loop with `switch`

Loops can work with `switch`.

Example:

```c
#include <stdio.h>

int main()
{
   int choice;
   int i;

   printf("1 - Count up\n");
   printf("2 - Count down\n");
   printf("Enter choice: ");
   scanf("%d", &choice);

   switch (choice)
   {
      case 1:
         for (i = 1; i <= 5; i++)
         {
            printf("%d\n", i);
         }
         break;

      case 2:
         for (i = 5; i >= 1; i--)
         {
            printf("%d\n", i);
         }
         break;

      default:
         printf("Invalid choice\n");
         break;
   }

   return 0;
}
```

The `switch` chooses what kind of loop to run.

# 20. Braces with loops

Use braces even when the loop has only one line.

This works:

```c
for (i = 1; i <= 5; i++)
   printf("%d\n", i);
```

But for learning, use this:

```c
for (i = 1; i <= 5; i++)
{
   printf("%d\n", i);
}
```

Braces make the code clearer and safer.

# 21. Common mistakes

## Mistake 1: forgetting update

Bad:

```c
int i = 1;

while (i <= 5)
{
   printf("%d\n", i);
}
```

Problem:

```text
i never changes
```

## Mistake 2: wrong comparison

Bad:

```c
for (i = 1; i < 5; i++)
{
   printf("%d\n", i);
}
```

Output:

```text
1
2
3
4
```

It does not print 5 because `i < 5` stops before 5.

Use this if you want 1 through 5:

```c
i <= 5
```

## Mistake 3: starting at wrong number

```c
for (i = 0; i <= 5; i++)
```

This prints:

```text
0
1
2
3
4
5
```

That may be correct later for arrays, but if the problem says 1 to 5, start at 1.

## Mistake 4: semicolon after loop

Bad:

```c
while (i <= 5);
```

Bad:

```c
for (i = 1; i <= 5; i++);
```

# 22. Mental model

Read this:

```c
for (i = 1; i <= 5; i++)
```

as:

```text
Start i at 1.
Keep going while i is less than or equal to 5.
After each loop, add 1 to i.
```

Read this:

```c
while (i <= 5)
```

as:

```text
As long as i is less than or equal to 5, keep looping.
```

# 23. What K&R 3.5 is mainly teaching

The main idea of K&R 3.5 is:

```text
C has loop statements that let you repeat code.
while is useful when repetition depends on a condition.
for is useful when initialization, condition, and update belong together.
```

Most beginner counting loops can be written with either `while` or `for`.
