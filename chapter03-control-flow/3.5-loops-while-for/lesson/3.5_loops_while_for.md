# 3.5 - Loops: While and For

## Big idea

A loop repeats code while a condition is true.

K&R 3.5 focuses on:

```text
while loop
for loop
comma operator in for loops
```

---

## While loop

A `while` loop checks the condition first.

Format:

```c
while (condition) {
    statements;
}
```

Example:

```c
int i = 1;

while (i <= 5) {
    printf("%d\n", i);
    i++;
}
```

Meaning:

```text
start i at 1
repeat while i <= 5
print i
increase i
stop when i becomes 6
```

---

## Three loop parts

Most loops have three parts:

```text
start value
condition
update
```

Example:

```c
int i = 1;

while (i <= 5) {
    printf("%d\n", i);
    i++;
}
```

Parts:

```text
int i = 1    start value
i <= 5       condition
i++          update
```

---

## Infinite loop warning

If the condition never becomes false, the loop never stops.

Bad:

```c
int i = 1;

while (i <= 5) {
    printf("%d\n", i);
}
```

Problem:

```text
i never changes
i stays 1 forever
i <= 5 is always true
```

Fix:

```c
i++;
```

Simple rule:

```text
If the loop condition depends on a variable, that variable usually needs to change inside the loop.
```

---

## For loop

A `for` loop puts the start, condition, and update in one line.

Format:

```c
for (start; condition; update) {
    statements;
}
```

Example:

```c
for (i = 1; i <= 5; i++) {
    printf("%d\n", i);
}
```

Parts:

```text
i = 1     start
i <= 5    condition
i++       update
```

---

## While and for can do the same thing

While version:

```c
i = 1;

while (i <= 5) {
    printf("%d\n", i);
    i++;
}
```

For version:

```c
for (i = 1; i <= 5; i++) {
    printf("%d\n", i);
}
```

Both print:

```text
1
2
3
4
5
```

---

## When to use while

Use `while` when the loop depends on a condition.

Example idea:

```text
repeat while not finished
repeat while input is valid
repeat while value is less than limit
```

---

## When to use for

Use `for` when counting is simple.

Example idea:

```text
print numbers 1 to 5
repeat exactly 10 times
go through array indexes later
```

---

## Empty parts in for loop

A `for` loop can have empty parts.

Example:

```c
for (;;) {
    printf("infinite loop\n");
}
```

This means infinite loop.

Do not use this yet unless you have a way to stop it later.

---

## Comma operator in for loops

K&R shows that a `for` loop can update more than one variable using commas.

Example:

```c
for (i = 0, j = 5; i < j; i++, j--) {
    printf("i = %d, j = %d\n", i, j);
}
```

Meaning:

```text
start i at 0 and j at 5
loop while i < j
increase i and decrease j each loop
```

This is useful later with arrays and strings.

## What I need to remember

```text
loop = repeat code

while checks condition first

for puts start, condition, and update in one line

most loops need:
start value
condition
update

forgetting the update can cause an infinite loop

while and for can do the same job

use while for condition-based loops

use for for simple counting loops

```

## Nested loops

A nested loop is a loop inside another loop.

Example:

```c
int row;
int col;

for (row = 1; row <= 3; row++) {
    for (col = 1; col <= 4; col++) {
        printf("*");
    }
    printf("\n");
}
```

Output:

```text
****
****
****
```

Meaning:

```text
outer loop = controls rows
inner loop = controls columns
```

Nested loops are useful for grids, tables, rows and columns, and later arrays.

Beginner rule:

```text
one loop = simple repetition
two nested loops = rows and columns
three nested loops = possible, but harder to read
```

---

## Comma operator in a for loop

A `for` loop can use commas to do more than one action in the start or update part.

Example:

```c
int i;
int j;

for (i = 0, j = 5; i < j; i++, j--) {
    printf("i = %d, j = %d\n", i, j);
}
```

The `for` loop has three parts:

```text
for (start; condition; update)
```

In this example:

```text
i = 0, j = 5   start two variables
i < j          condition
i++, j--       update two variables
```

The comma means:

```text
do the first expression, then do the second expression
```

So:

```c
i = 0, j = 5
```

means:

```text
set i to 0
set j to 5
```

And:

```c
i++, j--
```

means:

```text
increase i
decrease j
```

The comma operator is useful in `for` loops when two variables need to change together.

Beginner rule:

```text
Use commas in for loops only when it makes the loop clearer.
Do not overuse it.
```
