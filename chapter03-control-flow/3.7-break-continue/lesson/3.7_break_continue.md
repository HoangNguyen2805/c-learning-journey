# 3.7 - Break and Continue

## Big idea

`break` and `continue` change what happens inside a loop.

```text id="nyy1hl"
break = leave the loop early
continue = skip the rest of this loop cycle
```

They are useful when a loop should not always run normally from top to bottom.

---

## Break

`break` stops the loop immediately.

Example:

```c id="opq0w3"
int i;

for (i = 1; i <= 10; i++) {
    if (i == 5)
        break;

    printf("%d\n", i);
}
```

Output:

```text id="jexd7c"
1
2
3
4
```

When `i` becomes `5`, `break` leaves the loop.

---

## Continue

`continue` skips the rest of the current loop cycle and goes to the next cycle.

Example:

```c id="u756ie"
int i;

for (i = 1; i <= 5; i++) {
    if (i == 3)
        continue;

    printf("%d\n", i);
}
```

Output:

```text id="36py7v"
1
2
4
5
```

When `i` is `3`, `continue` skips the `printf`.

---

## Break vs continue

```text id="7zr7ue"
break = stop the whole loop

continue = skip this one loop cycle
```

Example:

```text id="fj47db"
break says: I am done with this loop.

continue says: skip this round, but keep looping.
```

---

## Break in switch

`break` is also used in `switch`.

```c id="5c7rho"
switch (choice) {
    case 1:
        printf("Start\n");
        break;

    case 2:
        printf("Quit\n");
        break;
}
```

In a switch, `break` stops fall-through.

---

## Continue works only in loops

`continue` belongs in loops:

```text id="3qgyq3"
while
for
do-while
```

Do not use `continue` inside a switch by itself.

---

## What I need to remember

```text id="6dygkw"
break stops the loop immediately

continue skips the rest of the current loop cycle

break can be used in loops and switch

continue is used in loops

break = leave

continue = skip to next cycle
```
