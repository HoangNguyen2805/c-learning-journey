# 3.3 - Else-If

## Big idea

`else if` lets a program choose from more than two paths.

Example:

```c
if (score >= 90)
    printf("A\n");
else if (score >= 80)
    printf("B\n");
else if (score >= 70)
    printf("C\n");
else
    printf("Fail\n");
```

---

## How it works

C checks the conditions from top to bottom.

```text
first condition true  -> run that part, skip the rest
first condition false -> check the next else if
none true             -> run else, if there is one
```

---

## Order matters

Put the strongest or most specific condition first.

Correct:

```c
if (score >= 90)
    printf("A\n");
else if (score >= 80)
    printf("B\n");
else if (score >= 70)
    printf("C\n");
else
    printf("Fail\n");
```

Wrong order:

```c
if (score >= 70)
    printf("C\n");
else if (score >= 80)
    printf("B\n");
else if (score >= 90)
    printf("A\n");
```

If `score` is `95`, the first condition `score >= 70` is already true, so the later checks never happen.

---

## Else is optional

You can write:

```c
if (score >= 90)
    printf("A\n");
else if (score >= 80)
    printf("B\n");
```

But if none of the conditions are true, nothing happens.

Using `else` gives a final fallback.

---

## Else-if vs separate if statements

This checks one chain:

```c
if (score >= 90)
    printf("A\n");
else if (score >= 80)
    printf("B\n");
else
    printf("Other\n");
```

Only one branch runs.

This checks separate conditions:

```c
if (score >= 90)
    printf("A\n");

if (score >= 80)
    printf("B\n");
```

More than one `if` can run.

---

## What I need to remember

```text
else if = check another condition if the previous condition was false

conditions are checked from top to bottom

only one branch of an if / else-if / else chain runs

order matters

else is the final fallback

separate if statements are not the same as an else-if chain
```
