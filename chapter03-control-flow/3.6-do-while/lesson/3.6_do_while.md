# 3.6 - Do-While

## Big idea

A `do-while` loop runs the body first, then checks the condition.

Format:

```c
do {
    statements;
} while (condition);
```

Important:

```text
The semicolon after while(condition) is required.
```

---

## Do-while loop

Example:

```c
int i = 1;

do {
    printf("%d\n", i);
    i++;
} while (i <= 5);
```

This prints:

```text
1
2
3
4
5
```

---

## Difference between while and do-while

A `while` loop checks before running.

```c
while (condition) {
    statements;
}
```

A `do-while` loop checks after running.

```c
do {
    statements;
} while (condition);
```

Simple rule:

```text
while = maybe runs zero times

do-while = always runs at least once
```

---

## Example: condition false at start

```c
int i = 10;

while (i < 5) {
    printf("while runs\n");
}
```

This does not run because `i < 5` is false.

But:

```c
int i = 10;

do {
    printf("do-while runs\n");
} while (i < 5);
```

This runs one time, then stops.

---

## When do-while is useful

Use `do-while` when something should happen at least once.

Example ideas:

```text
show a menu at least once
ask for input at least once
run one attempt before checking
repeat until user chooses quit
```

We will use input later. For now, use simple fixed variables.

---

## Important semicolon

Correct:

```c
do {
    printf("Hello\n");
} while (i < 5);
```

Wrong:

```c
do {
    printf("Hello\n");
} while (i < 5)
```

The final semicolon is required.

---

## What I need to remember

```text
do-while runs the body first

condition is checked after the body

do-while always runs at least once

while can run zero times

do-while needs a semicolon after while(condition);

do-while is useful for menus and input validation later
```

---

## While vs For vs Do-While

All three loops repeat code.

```text
while
for
do-while
```

The difference is mostly **where the start, condition, and update happen**.

---

## Similarities

All loops need a way to:

```text
start
check a condition
repeat code
eventually stop
```

If the condition never becomes false, the loop can become an infinite loop.

---

## While loop

Use `while` when the loop depends mainly on a condition.

```c
int i = 1;

while (i <= 5) {
    printf("%d\n", i);
    i++;
}
```

Best for:

```text
repeat while something is true
repeat until a condition changes
```

Important:

```text
while checks the condition before running
while can run zero times
```

---

## For loop

Use `for` when counting is simple and clear.

```c
int i;

for (i = 1; i <= 5; i++) {
    printf("%d\n", i);
}
```

Best for:

```text
counting loops
repeating a known number of times
array indexes later
```

Important:

```text
for puts start, condition, and update in one line
```

---

## Do-while loop

Use `do-while` when the loop body should run at least once.

```c
int i = 1;

do {
    printf("%d\n", i);
    i++;
} while (i <= 5);
```

Best for:

```text
menus later
input validation later
anything that must happen at least once
```

Important:

```text
do-while checks the condition after running
do-while always runs at least once
```

---

## Quick comparison

```text
Loop type    Condition checked       Runs at least once?     Best use

while        before the body         no                      condition-based loop

for          before the body         no                      counting loop

do-while     after the body          yes                     menu/input-style loop
```

---

## Same job, different style

These two do the same thing.

While version:

```c
int i = 1;

while (i <= 5) {
    printf("%d\n", i);
    i++;
}
```

For version:

```c
int i;

for (i = 1; i <= 5; i++) {
    printf("%d\n", i);
}
```

Both print:

```text
1
2
3
4
5
```
[IMPORTANCE]
The for statement:

```c
for (expr1; expr2; expr3) {
    statement
}
```

is equivalent to:

```c
expr1;
while (expr2) {
    statement
    expr3:
}
```

---

## What I need to remember

```text
while = condition first

for = counting loop with start, condition, update in one line

do-while = body first, condition after

while and for can run zero times

do-while always runs at least once

use for when counting is clear

use while when repeating depends on a condition

use do-while when something must happen at least once
```
