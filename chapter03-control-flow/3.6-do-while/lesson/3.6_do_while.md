# K&R 3.6 - `do-while` Loop

## Goal

This lesson explains the `do-while` loop.

By the end of this lesson, you should understand:

* what `do-while` does
* how it is different from `while`
* why it always runs at least once
* how to use it for menus
* how to use it for input validation
* where the semicolon goes
* common beginner mistakes

---

# 1. What is a `do-while` loop?

A `do-while` loop is a loop that checks the condition after the loop body.

Format:

```c
do
{
   statement1;
   statement2;
} while (condition);
```

Important:

```c
} while (condition);
```

The semicolon at the end is required.

---

# 2. Main idea

A `do-while` loop means:

```text
Do this code first.
Then check if we should do it again.
```

That means the body always runs at least one time.

---

# 3. Basic example

```c
#include <stdio.h>

int main()
{
   int i = 1;

   do
   {
      printf("%d\n", i);
      i++;
   } while (i <= 5);

   return 0;
}
```

Output:

```text
1
2
3
4
5
```

---

# 4. Step-by-step

Code:

```c
int i = 1;

do
{
   printf("%d\n", i);
   i++;
} while (i <= 3);
```

Step-by-step:

```text
i = 1
Run body first
Print 1
i becomes 2
Check: 2 <= 3 true

Run body again
Print 2
i becomes 3
Check: 3 <= 3 true

Run body again
Print 3
i becomes 4
Check: 4 <= 3 false

Stop
```

Output:

```text
1
2
3
```

---

# 5. Difference between `while` and `do-while`

## `while`

Checks condition first.

```c
while (condition)
{
   statements;
}
```

Meaning:

```text
Check first.
If true, run.
If false, skip.
```

## `do-while`

Runs first, checks later.

```c
do
{
   statements;
} while (condition);
```

Meaning:

```text
Run first.
Then check if it should repeat.
```

---

# 6. Example where the difference matters

## `while`

```c
#include <stdio.h>

int main()
{
   int i = 10;

   while (i <= 5)
   {
      printf("%d\n", i);
      i++;
   }

   return 0;
}
```

Output:

```text
No output
```

Because `10 <= 5` is false at the start.

## `do-while`

```c
#include <stdio.h>

int main()
{
   int i = 10;

   do
   {
      printf("%d\n", i);
      i++;
   } while (i <= 5);

   return 0;
}
```

Output:

```text
10
```

Why?

Because `do-while` runs the body once before checking the condition.

---

# 7. When to use `do-while`

Use `do-while` when the code must run at least once.

Common beginner examples:

```text
Show menu at least once.
Ask user for input at least once.
Repeat if input is invalid.
Repeat game/menu until user quits.
```

---

# 8. `do-while` for input validation

Example: keep asking until the user enters a positive number.

```c
#include <stdio.h>

int main()
{
   int number;

   do
   {
      printf("Enter a positive number: ");
      scanf("%d", &number);

      if (number <= 0)
      {
         printf("Invalid. Try again.\n");
      }

   } while (number <= 0);

   printf("You entered: %d\n", number);

   return 0;
}
```

Why `do-while` is good here:

```text
We need to ask the user at least once.
Then if the number is invalid, ask again.
```

---

# 9. Menu with `do-while`

A menu should usually show at least one time.

```c
#include <stdio.h>

int main()
{
   int choice;

   do
   {
      printf("\nMenu\n");
      printf("1 - Say hello\n");
      printf("2 - Say goodbye\n");
      printf("0 - Quit\n");
      printf("Enter choice: ");
      scanf("%d", &choice);

      switch (choice)
      {
         case 1:
            printf("Hello\n");
            break;

         case 2:
            printf("Goodbye\n");
            break;

         case 0:
            printf("Quitting...\n");
            break;

         default:
            printf("Invalid choice\n");
            break;
      }

   } while (choice != 0);

   return 0;
}
```

This menu repeats until the user enters `0`.

---

# 10. Why the condition is `choice != 0`

This:

```c
} while (choice != 0);
```

means:

```text
Keep looping while choice is not 0.
```

So:

```text
choice = 1 → repeat
choice = 2 → repeat
choice = 5 → repeat
choice = 0 → stop
```

---

# 11. `do-while` with `char`

When using `char` with `scanf`, use a space before `%c`.

Example:

```c
char again;

do
{
   printf("Run program\n");

   printf("Run again? Y/N: ");
   scanf(" %c", &again);

} while (again == 'Y');
```

The space in:

```c
scanf(" %c", &again);
```

skips leftover whitespace and newline.

Without the space, `%c` might read the leftover Enter key from the previous input.

---

# 12. Example: repeat while user says `Y`

```c
#include <stdio.h>

int main()
{
   char again;

   do
   {
      printf("This program is running.\n");

      printf("Run again? Y/N: ");
      scanf(" %c", &again);

   } while (again == 'Y' || again == 'y');

   printf("Program ended.\n");

   return 0;
}
```

Condition:

```c
again == 'Y' || again == 'y'
```

means:

```text
repeat if again is uppercase Y or lowercase y
```

# 13. Common beginner mistake: forgetting semicolon

Bad:

```c
do
{
   printf("Hello\n");
} while (choice != 0)
```

Missing semicolon.

Correct:

```c
do
{
   printf("Hello\n");
} while (choice != 0);
```

The `do-while` loop must end with a semicolon.

---

# 14. Common beginner mistake: putting condition in the wrong place

Bad:

```c
do (choice != 0)
{
   printf("Menu\n");
}
```

That is not the format.

Correct:

```c
do
{
   printf("Menu\n");
} while (choice != 0);
```

---

# 15. Common beginner mistake: updating the variable wrong

Bad:

```c
int i = 1;

do
{
   printf("%d\n", i);
} while (i <= 5);
```

Problem:

```text
i never changes
```

This causes an infinite loop.

Correct:

```c
int i = 1;

do
{
   printf("%d\n", i);
   i++;
} while (i <= 5);
```

---

# 16. `while` vs `do-while` decision rule

Use `while` when:

```text
The loop might need to run zero times.
```

Use `do-while` when:

```text
The loop must run at least one time.
```

Example:

```text
Validate user input → do-while
Menu program → do-while
Counting 1 to 10 → for
Unknown condition before start → while
```

---

# 17. Input validation example with range

Problem:

```text
Ask the user for a grade from 0 to 100.
Keep asking until valid.
```

Code:

```c
#include <stdio.h>

int main()
{
   int grade;

   do
   {
      printf("Enter grade 0-100: ");
      scanf("%d", &grade);

      if (grade < 0 || grade > 100)
      {
         printf("Invalid grade. Try again.\n");
      }

   } while (grade < 0 || grade > 100);

   printf("Valid grade: %d\n", grade);

   return 0;
}
```

Condition:

```c
grade < 0 || grade > 100
```

means:

```text
repeat while grade is less than 0 OR greater than 100
```

# 18. Combining `do-while`, `switch`, and `if`

Example:

```c
#include <stdio.h>

int main()
{
   int choice;
   int number;

   do
   {
      printf("\nMenu\n");
      printf("1 - Check even or odd\n");
      printf("0 - Quit\n");
      printf("Enter choice: ");
      scanf("%d", &choice);

      switch (choice)
      {
         case 1:
            printf("Enter number: ");
            scanf("%d", &number);

            if (number % 2 == 0)
            {
               printf("Even\n");
            }
            else
            {
               printf("Odd\n");
            }
            break;

         case 0:
            printf("Goodbye\n");
            break;

         default:
            printf("Invalid choice\n");
            break;
      }

   } while (choice != 0);

   return 0;
}
```

This uses:

```text
do-while → repeat the menu
switch   → choose menu action
if/else  → check even or odd
%        → remainder operator
```

# 19. Braces and indentation

Recommended style:

```c
do
{
   statements;
} while (condition);
```

Do not write messy code like this:

```c
do {
statements;
} while(condition);
```

This may compile, but it is harder to read.

Use clean indentation:

```c
do
{
   printf("Menu\n");
   scanf("%d", &choice);
} while (choice != 0);
```
